Web Development with JavaScript

Learn to create efficient and sustainable web applications.

Lesson 1: JavaScript Basics

Get started with variables and functions in JavaScript. 

let message = "Welcome to Tech Connects!";
function displayMessage() {
    console.log(message);
}
displayMessage();

Lesson 2: DOM Manipulation

Interact with web elements dynamically. 

document.addEventListener("DOMContentLoaded", function() {
    let element = document.createElement("p");
    element.textContent = "Sustainable web development!";
    document.body.appendChild(element);
});

Lesson 3: Event Handling

Respond to user actions for interactive sites. 

document.getElementById("myButton").addEventListener("click", function() {
    alert("You clicked for sustainability!");
});

Sustainability Project: Energy-Saving Web Tool

Create a tool to remind users to save energy. 

function remindEnergySave() {
    let reminder = confirm("Turn off unused devices to save energy!");
    if (reminder) {
        alert("Great choice for a sustainable future!");
    }
}
document.getElementById("energyButton").addEventListener("click", remindEnergySave);
