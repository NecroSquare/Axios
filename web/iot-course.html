Lesson 3: IoT Communication

Explore how IoT devices communicate data. 

# Pseudo-code: Sending Data
def send_data(data):
    if data['moisture'] < 50:
        return "Water the farm!"
    return "Conditions optimal"
print(send_data({'moisture': 40}))

Sustainability Project: Smart Farming Monitor

Create a simple IoT system to monitor farming conditions. 

# Smart Farming Monitor
def monitor_farm(temperature, moisture):
    if temperature > 35 or moisture < 40:
        return "Alert: Unfavorable conditions!"
    return "Farm conditions are good."

print(monitor_farm(36, 35))  # Output: Alert: Unfavorable conditions!
print(monitor_farm(25, 60))  # Output: Farm conditions are good.
