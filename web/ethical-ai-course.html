Lesson 2: Identifying Bias in AI

Learn to detect and mitigate bias in AI systems. 

# Pseudo-code: Checking Bias
data = {'gender': ['M', 'F', 'M', 'F'], 'score': [90, 85, 95, 80]}
if max(data['score']) - min(data['score']) > 10:
    print("Potential bias detected!")
else:
    print("No significant bias.")

Sustainability Project: Ethical AI Checklist

Create a checklist to ensure AI development is ethical. 

# Ethical AI Checklist
checklist = {
    'fairness': True,
    'transparency': True,
    'low_carbon': False
}
if all(checklist.values()):
    print("AI is ethically compliant!")
else:
    print("Review the following: ", [k for k, v in checklist.items() if not v])
